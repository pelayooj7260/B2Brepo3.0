# Future Website Updates & Enhancements

This checklist outlines all suggested features, refinements, and functional additions for the next phase of the website's evolution. As discussed, we are focusing on frontend, UI, and SEO enhancements, omitting backend databases and production hosting setups.

## 1. Additional Content Sections
- [x] **Social Proof / Client Logos:** Add an elegant, infinitely scrolling marquee of trusted company logos just below the hero section.
- [x] **Testimonials Carousel:** Implement a slider featuring quotes from satisfied clients, styled as clean `GlassCard` components.
- [x] **Interactive Case Studies:** Create a "Before & After" slider visually demonstrating a manual workflow vs. the sleek automated solution.
- [x] **FAQ Section:** Add an accordion-style Frequently Asked Questions section at the bottom of the page to handle common client objections before they submit a form.

## 2. SEO & Analytics Optimization
- [x] **Technical SEO:** Add `react-helmet-async` to manage dynamic `<title>` and `<meta>` description tags for better search engine ranking.
- [x] **Sitemap & Robots.txt:** Generate a `sitemap.xml` and `robots.txt` for proper Google indexing, using a placeholder domain for now.

## 3. Design & UI Polish
- [x] **Custom Floating Cursor:** Implement a custom, minimalist circular cursor that smoothly follows the mouse and dynamically expands when hovering over clickable elements. (Disabled on mobile).
- [x] **Light/Dark Mode Toggle:** Implement an alternative "Sunlit Shallows" color scheme (a light mode counterpart to Tranquil Depths) with a sleek, animated theme toggle in the navbar.
- [x] **Mobile Performance Tuning:** Profile and optimize the `framer-motion` animations to ensure perfect scrolling on mobile hardware.
- [x] **Custom 3D/Lottie Assets:** Replace standard icons with refined animations to elevate the premium SaaS feel even further. (Mock placeholders implemented).
